# 🚀 NOFX 一键部署到 Hugging Face - 快速指南

## ⚡ 5 分钟快速部署

### 第一步：创建 Space

1. 访问 https://huggingface.co/spaces
2. 点击 **"Create new Space"**
3. 填写信息：
   - **Space name**: `your-username/nofx`
   - **SDK**: 选择 **Docker**
   - **Hardware**: 选择 **CPU basic**
4. 点击 **"Create Space"**

### 第二步：准备文件

在项目目录执行：

```bash
# 复制 Dockerfile
cp Dockerfile.hf Dockerfile
```

### 第三步：上传代码

**方式 1：Git 推送（推荐）**

```bash
# 添加 remote
git remote add hf https://huggingface.co/spaces/YOUR_USERNAME/YOUR_SPACE_NAME

# 推送代码
git push hf main
```

**方式 2：Web 上传**

1. 在 Space 页面点击 **"Files and versions"**
2. 点击 **"Add file"** → **"Upload files"**
3. 上传所有文件（包括 `Dockerfile`）

### 第四步：配置环境变量

在 Space 的 **Settings** → **Variables** 中添加：

```
NOFX_ADMIN_PASSWORD = your-password-here
NOFX_TIMEZONE = Asia/Shanghai
AI_MAX_TOKENS = 4000
```

### 第五步：等待构建

- 构建需要 5-15 分钟
- 构建完成后自动启动
- 访问 Space URL 即可使用

## 🎯 首次配置

1. **添加 AI 模型**
   - 进入 "AI Models" 菜单
   - 添加 DeepSeek 或 Qwen API 密钥

2. **添加交易所**
   - 进入 "Exchanges" 菜单
   - 添加交易所 API 密钥

3. **创建交易员**
   - 进入 "Traders" 菜单
   - 创建新交易员并启动

## 🐛 遇到问题？

1. **构建失败** → 检查 `Dockerfile` 是否正确
2. **无法启动** → 查看日志，检查环境变量
3. **API 错误** → 确认后端服务已启动

详细故障排除：查看 `DEPLOY_HF_TUTORIAL.md`
