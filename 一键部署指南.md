### 第二步：从 GitHub 同步代码

**使用 Git 推送（最简单）：**

```bash
# 1. 克隆您的仓库
git clone https://github.com/tanzeking/nofx-12.git
cd nofx-12

# 2. 添加 Hugging Face Space 作为远程仓库
git remote add hf https://huggingface.co/spaces/YOUR_USERNAME/YOUR_SPACE_NAME

# 3. 推送代码
git push hf main
```

**或者直接在 Space 中上传：**
- 在 Space 页面点击 **"Files and versions"**
- 点击 **"Add file"** → **"Upload files"**
- 上传所有文件

### 第三步：准备 Dockerfile

在 GitHub 仓库中重命名：

```bash
# 克隆仓库
git clone https://github.com/tanzeking/nofx-12.git
cd nofx-12

# 重命名 Dockerfile
git mv Dockerfile.hf Dockerfile

# 推送更改
git commit -m "Add Dockerfile for HF"
git push origin main
```

或者在 Space 中直接创建 `Dockerfile`（复制 `Dockerfile.hf` 内容）

### 第四步：配置环境变量

在 Space Settings → Variables 添加：
- `NOFX_ADMIN_PASSWORD` = 您的密码
- `NOFX_TIMEZONE` = `Asia/Shanghai`

### 第五步：等待自动构建

- Hugging Face 会自动检测并构建
- 构建完成后即可使用

## 🎯 首次配置

1. **添加 AI 模型**
   - 进入 "AI Models" 菜单
   - 添加 DeepSeek 或 Qwen API 密钥

2. **添加交易所**
   - 进入 "Exchanges" 菜单
   - 添加交易所 API 密钥

3. **创建交易员**
   - 进入 "Traders" 菜单
   - 创建新交易员并启动

## 🐛 遇到问题？

1. **构建失败** → 检查 `Dockerfile` 是否正确
2. **无法启动** → 查看日志，检查环境变量
3. **API 错误** → 确认后端服务已启动

详细故障排除：查看 `DEPLOY_HF_TUTORIAL.md`
